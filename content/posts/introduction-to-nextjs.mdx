## What is NextJs?

NextJs is a React framework that allows you to build server-rendered React 
applications with ease. It provides a number of features out of the box, such

## Why use NextJs？

NextJs is a great choice for building server-rendered React applications beca it provides a number of features that make it easy to build fast and scalable applications. Some of the key features of Nexts include:
- Server-side rendering: NextJs allows you to render your React components or the server, which can improve performance and SEO.
- Static site generation: NextJs can generate static HTML files for your page at build time, which can improve performance and reduce hosting costs.
- API routes: NextJs allows you to create API routes that can be used to fetc data from external APIs or perform server-side logic.
